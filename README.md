# 바이드럼(Bidrum)
**바이드럼**(**Bidrum** in english) is a rhythm game with janggu(korean traditional drum)-style controller.

## How to build?
1. Install Godot engine with mono support
1. Build it

## Supported audio/video file formats
 - Video: Ogg Theora (`.ogv`)
 - Audio: WAV, Ogg Vorbis, MP3
